# Korea Inside Project Memory

## 목적

프로젝트에서 이미 확정된 사항은
다시 토론하거나 변경하지 않는다.

변경이 필요한 경우
변경 사유와 승인 기록을 남긴다.

---

## Memory Level

### Level A (Locked)

절대 변경하지 않는 결정

예)

- 프로젝트명

- 도메인

- 브랜드

- 디자인 철학

- AI 원칙

- SEO 철학

변경 시 반드시 승인 필요.

---

### Level B (Approved)

확정되었지만
향후 개선 가능한 사항

예)

페이지 구조

컴포넌트

FAQ

이미지 정책

---

### Level C (Draft)

실험 단계

언제든 수정 가능.

---

## 기록 방식

Decision

Reason

Date

Approved By

---

## 변경 규칙

Level A

↓

반드시 승인

Level B

↓

Diff

승인

Level C

↓

자유 수정 가능

## AI 규칙

AI는

이미 확정된 사항을

다시 제안하지 않는다.

동일한 논의를 반복하지 않는다.

기존 결정을 우선 참조한다.

---

## 최종 원칙

프로젝트는

결정을 계속 바꾸는 것이 아니라

결정을 계속 쌓아가는 것이다.
