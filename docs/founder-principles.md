# Korea Inside Founder Principles

------------------------------------------------

우리는

트래픽을 위해

신뢰를 포기하지 않는다.

------------------------------------------------

우리는

수익을 위해

사용자를 속이지 않는다.

------------------------------------------------

우리는

광고보다

문제 해결을 먼저 만든다.

------------------------------------------------

우리는

예약 사이트가 되지 않는다.

------------------------------------------------

우리는

한국을 이해하도록 돕는다.

------------------------------------------------

우리는

사용자가

스스로 선택하도록 만든다.

------------------------------------------------

우리는

사실과

공식 자료를 우선한다.

------------------------------------------------

우리는

AI를

생산성을 높이는 도구로 사용한다.

------------------------------------------------

우리는

새 기능보다

기존 페이지 완성도를 높인다.

------------------------------------------------

우리는

사용자의 시간을

존중한다.

------------------------------------------------

우리는

검색 엔진보다

사람을 먼저 생각한다.

------------------------------------------------

우리는

데이터를 통해 배우지만

가치를 버리지는 않는다.

------------------------------------------------

우리는

5년 후에도

후회하지 않을 결정을 한다.

------------------------------------------------

최종 원칙

Korea Inside는

돈을 버는 사이트가 아니라

신뢰를 쌓는 플랫폼이다.

신뢰가 쌓이면

수익은 자연스럽게 따라온다.
