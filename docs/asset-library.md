# Korea Inside Asset Library Standard

## 목적

프로젝트의 모든 재사용 가능한 자산을
검색 가능하고
추적 가능하며
재사용 가능한 형태로 관리한다.

Asset Library는 단순 파일 목록이 아니라
Korea Inside의 장기 운영 자산을 관리하는 기준이다.

관리 원칙

------------------------------------------------

- 실물 정확성이 중요한 자산은 사용자 제공 사진 또는 공식 자료를 우선한다.
- 핵심 정보는 이미지 안에만 넣지 않고 HTML Text로 유지한다.
- 재사용 가능한 자산은 중복 생성하지 않는다.
- 출처, 사용 페이지, 상태를 기록한다.
- 사용하지 않는 자산은 즉시 삭제하지 않고 Deprecated 또는 Archived로 표시한다.
- AI 생성 자산은 실사/공식 UI/실제 카드/지폐/앱 화면을 대체하지 않는다.
