# Korea Inside Page Template Standard

## 문서 성격

이 문서는 Korea Inside 페이지를 만들거나 검토할 때 빠르게 확인하는 최소 페이지 골격 체크리스트이다.

- 상세 페이지 구조와 흐름은 `docs/golden-page-template.md`를 따른다.
- 재사용 UI 블록과 컴포넌트 기준은 `docs/component-library.md`를 따른다.
- 이 문서는 HTML, CSS, JavaScript, 이미지 자산, 새 파일 생성, 기존 파일 수정을 자동 승인하지 않는다.
- 실제 파일 변경, 이미지 생성/추가, 자산 추가는 현재 사용자 명시 승인 범위 안에서만 수행한다.

## 공통 규칙

- 핵심 정보는 HTML 텍스트가 먼저 설명하고 이미지는 보조한다.
- HTML 텍스트를 유지한다.
- 대표 이미지와 AI 이미지는 필수 조건이 아니다.
- 자동 번역 가능해야 한다.
- 모바일 우선 설계한다.
- 긴 문단을 피한다.
- 사용자가 스스로 선택하도록 한다.
- 객관성을 유지한다.
- 비교를 남발하지 않는다.
