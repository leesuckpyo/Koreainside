## 공통 규칙

- 이미지가 먼저 설명한다.
- HTML 텍스트를 유지한다.
- 자동 번역 가능해야 한다.
- 모바일 우선 설계한다.
- 긴 문단을 피한다.
- 사용자가 스스로 선택하도록 한다.
- 객관성을 유지한다.
- 비교를 남발하지 않는다.
