# Korea Inside Project Charter

------------------------------------------------

Project Name

------------------------------------------------

Korea Inside

------------------------------------------------

Tagline

------------------------------------------------

How Korea Works

------------------------------------------------

Mission

------------------------------------------------

외국인이

한국을 더 쉽게 이해하도록 돕는다.

------------------------------------------------

Vision

------------------------------------------------

세계에서 가장 신뢰받는

한국 여행 문제 해결 플랫폼

------------------------------------------------

Core Promise

------------------------------------------------

우리는

관광지를 소개하지 않는다.

한국을 이해하도록 돕는다.

------------------------------------------------

Core Values

------------------------------------------------

Truth before Marketing

Trust before Revenue

Understanding before Recommendation

Quality before Quantity

Consistency before Speed

Long-term before Short-term

------------------------------------------------

Product Identity

------------------------------------------------

Korea Inside는

예약 사이트가 아니다.

관광 블로그가 아니다.

광고 사이트가 아니다.

한국 사용설명서이다.

------------------------------------------------

User Promise

------------------------------------------------

사용자는

정보를 읽는 것이 아니라

한국을 미리 경험한다.

------------------------------------------------

AI Promise

------------------------------------------------

AI는

프로젝트를 대신 결정하지 않는다.

프로젝트를 더 잘 만들도록 돕는다.

------------------------------------------------

Success Metrics

------------------------------------------------

사용자가

"이 사이트 덕분에 한국을 이해했다."

라고 느끼는 것.

------------------------------------------------

Never Change

------------------------------------------------

아래 항목은
프로젝트의 정체성이므로
특별한 승인 없이는 변경하지 않는다.

- Project Name
- Mission
- Vision
- Tagline
- Core Values
- Product Identity

------------------------------------------------

Final Principle

------------------------------------------------

Korea Inside는

웹사이트를 만드는 프로젝트가 아니다.

외국인이 한국을 이해하는 방식을
새롭게 만드는 프로젝트이다.
