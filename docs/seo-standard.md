# Korea Inside SEO Standard

## 1. 기본 원칙

- SEO는 Google 기준으로 설계한다.
- 관광지 소개보다 문제 해결 검색을 우선한다.
- 키워드 남용을 금지한다.
- 사용자가 검색하는 질문에 답하는 구조를 사용한다.

---

## 2. URL

규칙

- 소문자 사용
- 하이픈(-) 사용
- 짧고 명확하게 작성

예)

arrival-guide

tmoney-card

wowpass

esim-korea

korean-money

---

## 3. Title

- 60자 이내
- 핵심 키워드 포함
- 자연스럽게 작성

예)

How To Use WOWPASS In Korea

How To Buy A T-money Card

Best eSIM Guide For Korea

---

## 4. Meta Description

- 150~160자
- 문제 해결 중심
- 클릭을 유도하되 과장 금지

---

## 5. Heading

H1 : 1개

H2

H3

순서 유지

---

## 6. HTML

- 핵심 내용은 HTML Text
- 이미지 안에 핵심 정보 금지
- 자동 번역 가능 구조 유지

---

## 7. FAQ

- FAQ Schema 적용 가능하도록 작성
- 실제 질문 중심
- 중복 질문 금지

---

## 8. Internal Link

각 페이지는

3개 이상

관련 페이지 연결

---

## 9. Image SEO

- WebP 사용
- alt 작성
- 의미 있는 파일명 사용

예)

wowpass-card.webp

tmoney-machine.webp

---

## 10. Structured Data

가능한 경우

- FAQ
- Breadcrumb
- Article
- Organization

---

## 11. 콘텐츠 품질

- 쉬운 영어(B1~B2)
- 짧은 문단
- 객관적인 표현
- 최신 정보 유지

---

## 12. 금지 사항

- 키워드 반복
- 과장 표현
- 클릭베이트
- 의미 없는 페이지 생성

---

## 공개 기준

- SEO Checklist 통과
- Review Checklist 통과
- Broken Link 없음
- Mobile Friendly 확인
- Core Web Vitals 확인
